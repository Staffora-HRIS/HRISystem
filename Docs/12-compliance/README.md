## Overview

Staffora is a UK-only HRIS platform. All compliance documentation targets UK employment legislation including:

- **Employment Law** -- Right to work checks, employment contracts, disciplinary/grievance (ACAS), flexible working
- **Statutory Pay** -- SSP, SMP, SPP, ShPP, SAP calculations and HMRC reporting
- **Leave Entitlements** -- Holiday entitlement (Working Time Regulations), family leave, parental leave
- **Pensions** -- Auto-enrolment compliance, contribution management
- **GDPR/Data Protection** -- DSAR handling, data erasure, breach notification, consent management, privacy notices
- **Equality & Reporting** -- Gender pay gap reporting, equality and diversity monitoring
